# Magic
## 命名規則

**型名(クラス、構造体、列挙体)**  
パスカルケース  
名詞  
例：GameObject  

**変数名**  
小文字スネーク  
名詞  
メンバ変数はアンダースコア終わり  
例：color_state  

**定数名**  
大文字スネーク  
名詞  
例：LEMON_NUM  

**関数名**  
パスカルケース  
動詞  
例：LemonCreate  

**フォルダ名**  
パスカル  

**Material**  
パスカル  
アタッチするゲームオブジェクト名  


## 作業の仕方

Workフォルダを作成してあるのでそこに自分の名前のシーン名を作り、そこで作業を行ってください。  
例：arai  


ゲームオブジェクトは必ずプレハブ化してください。  
EmptyObject、プレハブのPositionは必ずx:0、y:0、z:0にしてください。  

## 注意点  
**Unityのバーション**  
5.1.0f3のバージョンで作業を行ってください
